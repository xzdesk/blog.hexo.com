    Centos7防火墙firewalld管理

    centos centos7 firewalld 防火墙
    发布时间 : 2023-07-24 10:37
    阅读 :
    1. firewalld 防火墙管理
    2. SELinux管理

    firewalld 防火墙管理

    systemctl status firewalld  查看防火墙状态
systemctl disable firewalld 关闭开机自启动
systemctl enable firewalld  开启开机自启动
systemctl stop firewalld    关闭防火墙

# 或增加防火墙开放端口
firewall-cmd --permanent --add-port=2377/tcp
firewall-cmd --reload

    SELinux管理

    # 查看状态
sestatus
/usr/sbin/sestatus -v
getenforce

# 关闭防火墙,生效需要重启
setenforce 0
sed -i 's/SELINUX=enforcing/SELINUX=disabled/' /etc/selinux/config

# 临时关闭
setenforce 0    # 设置SELinux 成为permissive模式
setenforce 1    # 设置SELinux 成为enforcing模式

    注意：改变SELinux状态后，记得重启服务器
